# User Stories

1. **As a Maintenance Technician**, I can access the app to check if any machines have issues so that I can go and fix them.

2. **As a Maintenance Director**, I can access the app to see if any machine has been experiencing frequent failures so that I can investigate whether there is a more serious underlying problem.

3. **As a Manager**, I can access the app to check whether a machine is causing financial losses due to frequent breakdowns.

4. **As an Administrator**, I can access the app to verify whether maintenance technicians are performing their tasks and whether there are many machines currently not operational.

5. **As a Maintenance Technician**, I can access the app to see if any machine shows signs of a potential future failure.

6. **As a Maintenance Technician**, I can access the app to view the priority of each machine based on its importance, downtime duration, and severity of failure.

7. **As a Maintenance Technician**, if I have no assigned tasks, I can check the app to see whether a colleague is experiencing difficulties and assist them.

8. **As a Maintenance Technician**, if I realize that a machine cannot be repaired alone, I can flag it in the app so that my colleagues receive a notification.

9. **As a Maintenance Director**, I can access the app to see which technician has repaired the most machines.

10. **As a Manager or Administrator**, when a new machine is acquired, I can register it in the app.

11. **As a Manager or Administrator**, when a machine is removed, I can delete it from the app.

12. **As a Manager, Administrator, or Maintenance Director**, I can view previously used machines and access their maintenance history in the app.
