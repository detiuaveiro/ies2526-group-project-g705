# Functional Requirements

  ### FR1 –
        The system shall support different types of users.

   ### FR2 –
        The system shall authenticate users before allowing access to system functionalities.

   ### FR3 –
        The system shall restrict and allow access to certain functionalities based on the user type.

   ### FR4 –
        The system shall store machine information (e.g., identification, status, and operational data).

   ### FR5 –
        The system shall allow machine information to be updated.

   ### FR6 –
        The system shall collect data from vibration, pressure, and temperature sensors.

   ### FR7 –
        The system shall store historical sensor data for future analysis.

   ### FR8 –
        The system shall automatically generate an issue record when a failure is detected.

   ### FR9 –
        The system shall record maintenance activities performed and those that are still pending.

   ### FR10 –
        The system shall allow maintenance technicians to send requests.
